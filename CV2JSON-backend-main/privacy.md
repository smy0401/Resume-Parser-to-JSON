# Privacy Policy — CV 2 JSON

**Last Updated:** October 2025

This service processes uploaded CVs for JSON extraction in-memory and does **not persist user files** or personally identifiable information (PII).

## Data Handling
- Uploaded files are stored temporarily on disk for parsing.
- Temporary files are deleted immediately after processing.
- No data is shared, logged, or transmitted externally.

## Logging
- Logs can be configured to **redact personal identifiers** (emails, phone numbers, names) via `REDACT_PII=true` (default).
- Only non-sensitive metadata (latency, request ID, etc.) is retained.

## Storage & Retention
- No data is permanently stored.
- No persistent database or filesystem traces remain after request completion.

## Contact
For compliance or deletion requests, contact: `privacy@yourdomain.com`
